MongohqryApi

This is the MongohqryApi API

getTables

List the names of the available tables in this storage. By default, all tables are listed, use 'names' parameter to get specific tables. Use 'include_properties' to include any properties of the tables.

URL

https://your-dsp-name.dreamfactory.com/mongohqry

HTTP Method

GET

Response Type

Tables

Parameters

    include_properties : boolean (query)
    optional

    Return all properties of the tables, if any.

names : string (query)
optional

Comma-delimited list of the table names to retrieve.

deleteTables

Set the names of the tables to delete or set 'force' to true to clear the database.Alternatively, to delete by table definitions or a large list of names, use the POST request with X-HTTP-METHOD = DELETE header and post array of definitions or names.

URL

HTTP Method

Response Type

Parameters

    optional

    Comma-delimited list of the table names to delete.

    force : boolean (query)
    optional

    Set force to true to delete all tables in this database, otherwise 'names' parameter is required.

getRecords

Use the 'ids' or 'filter' parameter to limit resources that are returned. Use the 'fields' parameter to limit properties returned for each resource. By default, all fields are returned for all resources. Alternatively, to send the 'ids' or 'filter' as posted data use the POST request with X-HTTP-METHOD = GET header and post array of ids or a filter.

URL

HTTP Method

Response Type

    optional

    limit : int (query)
    optional

    Set to limit the filter results.

    offset : int (query)
    optional

    Set to offset the filter results to a particular record count.

    order : string (query)

    SQL-like order containing field and direction for filter results.

    fields : string (query)
    optional

    Comma-delimited list of field names to retrieve for each record.
